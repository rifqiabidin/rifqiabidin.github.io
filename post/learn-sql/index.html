Belajar SQL

Nov 6, 2024

Jika kamu membaca berbagai deskripsi di lowongan pekerjaan yang terkait dengan data, hampir bisa dipastikan kamu akan menemukan SQL dan bahasa pemrograman sebagai syaratnya. Kemampuan ini merupakan kemampuan dasar yang saya pikir harus dikuasai terlepas dari role pekerjaannya, skill ini akan berguna setidaknya untuk pengambilan dan pengolahan data dari database.

Disclaimer: saya menggunakan pos PostgreSQL Tutorial dari Neon sebagai referensi

  13. Tipe Data
